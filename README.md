# WeatherCrawler
爬取天气数据的爬虫

运行之前，确保已经安装scrapy、mongodb


安装scrapy: <code>pip install scrapy</code>

# 运行爬虫
<code>scrapy crawl province_spider</code>

会依次从各省、市、县爬取天气数据，存入数据库，最终的数据如下：
